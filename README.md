# StudentMessComplaintApp

Imagine you love your home-cooked meals but, unfortunately, you’re staying in a hostel. We all know the reputation of hostel food—often unappealing, and despite multiple complaints to the chef or workers, the quality rarely improves. Sometimes, it even gets worse.

My project, StudentMessComplaintApp, focuses on addressing this issue. In this app, users first need to sign up and then log in to verify their accounts. Once logged in, they can raise a complaint about the food. Each complaint is stored in a text file, and if another user submits a complaint, it is appended to the same file.

The main program is connected to the administration department. This ensures that every complaint made by the users is sent directly to the administration office for review and action.

This is the core idea of my project, aiming to improve the communication process and ultimately the quality of hostel food.
